WP Help Popup plugin have valuable features for your WordPress Website. Users could reach you through Live Chat(Pro), Whatsapp Chat or they could send their valuable feeedback with their suggestions or idea. 

== Description ==

WP Help Popup plugin provide you quick help within your website. Its having quite rich features like- Whats App chat Integration and show recent posts into help box. There is search box into popup for search sepecific posts. You could display help box into specific pages/post types or across website.

Packed with loads of features, WP Help Popup is ready to go out of the box. It’s also extensible, easy to use, and completely customizable.

== Features ==

*   Show What'sApp Chat box
*   Show Support Call box
*   Show recent posts into help box.
*   Enable/disable help box into different part of website.
*   Enable help box into specific pages/post types.
*   Ajax search box into helpbox and show results within helpbox.
*   Enable/Disable search box into helpbox.
*   All frontend labels/icons are manageable from plugin setting page.

== Installation ==
   
1. Log-in to your WordPress admin panel. 
2. Go to plugins and click on Add New button.
3. Upload plugin's Zip file from your PC and activate plugin.
4. Now You could see "WP Help Popup" into dashboard menu.

= What's the difference between WP Help Popup and WP Help Popup Pro? =

WP Help Popup is our cornerstone plugin, and it’s have search into website popup, Whatsapp floating Icon, Customer care call box and more.

In other words, WP Help Popup Pro runs alongside WP Help Popup and enhances it with extra features, including most important Live Chat, Multiple Whatsapp Account List , feedback submission, articles listing, drop message feature without Login and more.
